# 交通数据挖掘工具箱

本仓库是一个基于 Python 的交通数据挖掘工具箱，主要用于交通数据预处理、特征工程、聚类分析、自然语言处理和结果可视化。
其中的代码主要来源于本人在交通出行行为分析、MaaS 数据分析和机器学习建模等研究与课程项目中的实践积累。

## 项目简介

交通数据通常具有来源多样、结构复杂、噪声较多等特点，常见数据包括出行记录、站点信息、轨迹数据、调查数据和用户行为特征等。
本工具箱整理了交通数据分析中常用的一些处理模块，可用于原始数据清洗、异常值处理、行为特征提取、聚类分析、不平衡样本处理、文本建模和结果可视化等任务。

本仓库更偏向于“可复用代码工具集合”，而不是单一完整系统。每个文件夹对应一个相对独立的数据处理或建模任务。

## 主要功能

### 1. 数据清洗与预处理

* 基于 2.5IQR 方法清洗极端值
* 提取特定行数据
* 缺失值与异常样本处理
* 欠采样删除数据

### 2. 特征工程

* 统计出行次数
* 统计用户层面的出行次数
* 计算 80% 分位值特征
* VIF 多重共线性分析
* PCA 降维处理

### 3. 聚类与机器学习

* DBSCAN 聚类
* HDBSCAN 聚类
* AP 聚类
* SMOTE 重采样
* 不平衡样本处理

### 4. 交通数据处理

* GCJ-02 坐标转换
* 公交上下站点还原
* 调用高德 API 计算距离

### 5. 自然语言处理

* 使用 jieba 进行中文分词
* LDA 主题模型建模
* 将结构化出行记录转换为自然语言描述

### 6. 可视化分析

* 雷达图绘制
* 聚类结果画像展示
* 多维特征对比分析

## 典型应用场景

### 1. 出行行为聚类分析

可基于用户出行次数、出行距离、出行时间、交通方式使用频率等特征，对用户进行聚类分析，识别不同类型的出行行为模式。

典型流程如下：

```text
原始出行记录
→ 数据清洗
→ 特征提取
→ 标准化处理
→ 聚类分析
→ 用户群体画像
→ 结果可视化
```

### 2. 交通行为特征工程

可用于计算用户出行次数、分位值特征、多重共线性指标等，为后续聚类、分类、回归或统计建模提供输入特征。

### 3. 结构化出行数据的自然语言转换

可将结构化出行记录转换为自然语言描述，为后续文本建模、语义分析或大语言模型相关任务提供基础数据。

## 使用方法

大多数文件夹中的代码是相互独立的 Python 脚本。一般使用流程如下：

```bash
# 1. 克隆本仓库
git clone https://github.com/Liaott777/transport-data-mining-toolkit.git

# 2. 进入项目目录
cd transport-data-mining-toolkit

# 3. 安装常用依赖
pip install pandas numpy scikit-learn matplotlib jieba

# 4. 进入对应文件夹，运行相应脚本
python your_script_name.py
```

部分脚本需要根据本地环境修改数据路径、文件名或 API Key。
